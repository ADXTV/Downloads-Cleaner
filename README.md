# 🗑 Downloads Cleaner
Скрипт для автоматической сортировки папки **Downloads**

## Возможности
- ✅ Автоматическое создание папок
- ✅ Сортировка 20+ типов файлов

## 🚀 Быстрый старт
```bash
# Склонируйте репозиторий
git clone https://github.com/ADXTV/Downloads-Cleaner
# Запустите скрипт
python main.py
```
